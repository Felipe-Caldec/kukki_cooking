Se realiza una página web para una pastelería
La página se realiza en base a los colores de la pastelería y tiene un diseño responsivo, pensada fundamentamente en dispositivos móviles
Cuenta con tarjetas que muestran los distintos productos. En ellas se puede ver una imágen del producto, su descripción y precio. 
Además, posee un botón detalle que permite agrandar la imagen y ver más detalles del producto
Las cartas con los productos pueden ser filtradas según el tipo de producto
Los productos pueden ser agregados y quitados de un carrito de compras
Al presionar finalizar compra lleva a otra ventana donde se resume el pedido, se muestra el total de la compra 
Junto con ello se presenta un formulario de contacto el que envía esa información a un correo predeterminado para poder contactar al cliente confirmar el pedido
y coordinar el envío 
Una de las mayores innovaciones que puede realizar en esta página fue el empotrar publicaciones de instagram que al presionarlas te dirigen a la red social
